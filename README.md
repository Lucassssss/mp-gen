```bash

cp apps/api/.env.example apps/api/.env.local
```

з„¶еҗҺзј–иҫ‘еҗ„ `.env.local` ж–Үд»¶пјҢеЎ«е…ҘжӮЁзҡ„й…ҚзҪ®пјҡ

